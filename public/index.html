About Resume Tracker

Resume Tracker is a modern, full-stack application designed to simplify resume management. Built with Node.js, Express, and Azure cloud services, it provides a seamless experience for both job candidates and recruiters.

Technologies Used: Node.js, Express.js, Azure Cosmos DB, Azure Blob Storage
